Getting Started With Golf
=========================

1. Compile golf.jar:
        
        cd &lt;golf directory&gt;
        ant

2. Start the server:

        cd &lt;golf directory&gt;
        ./golf-start.sh -p &lt;port&gt; appname@path_to_approot [...] 

3. Access the application: point your browser to (http://hostname:port/appname/ "http://hostname:port/appname/").
