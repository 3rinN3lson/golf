Getting Started With Golf
=========================

###1. Compile

In a terminal:
        
        cd &lt;golf directory&gt;
        ant

###2. Start the server

In a terminal:

        cd &lt;golf directory&gt;
        ./golf-start.sh -p &lt;port&gt; appname@path_to_approot [...] 

###3. Access the application
3. Point your browser to http://hostname:port/appname.
